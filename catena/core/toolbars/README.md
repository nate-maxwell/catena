# Directory containing all toolbars.

## Client Toolbar
The file / edit / view / help toolbar at the top of the client.

## Status Toolbar
The status toolbar at the bottom of the client, providing information and feedback
to the user.

## Actions Toolbar
The shelf-like toolbar with buttons to create nodes.
This is the primary node gallery for the application.
Each tab is its own toolbar located in the `action_sub_bars` directory.
